Pika Macro ASsembler
originally by darkfader

A simple 2-pass assembler with a customizable instruction set.

Executables:
	pmas					assembler
	pmdis					disassembler

Further information:
	test/readme.S			a quick reference of directives and required syntax
	cpu/**.s				CPU specific stuff
